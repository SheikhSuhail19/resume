# Resume — Sheikh Suhail Khursheed

LaTeX source for my professional resume.

## Build

### Using Overleaf (easiest)
1. Upload `SheikhSuhailKhursheed.tex` to [Overleaf](https://www.overleaf.com)
2. Compile and download the PDF

### Local build
```bash
pdflatex SheikhSuhailKhursheed.tex
```

Or with latexmk:
```bash
latexmk -pdf SheikhSuhailKhursheed.tex
```
